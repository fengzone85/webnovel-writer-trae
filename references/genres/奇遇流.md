# 奇遇流题材模板

## 题材特征

### 世界观设定
- 都市/异界
- 奇遇连连
- 随机事件

### 核心元素
- 奇遇获得
- 宝藏发现
- 高人指点
- 快速成长

### 常见金手指
- 奇遇系统
- 随机奖励
- 师傅相助

## Strand 比例建议
- Quest线: 60%（奇遇、探险）
- Fire线: 20%（奇遇缘分）
- Constellation线: 20%（世界设定）

## 爽点类型
1. 获得奇遇
2. 发现宝藏
3. 高人收徒
4. 传承获得
5. 顿悟突破

## 钩子类型
1. 遗迹开启
2. 深山奇遇
3. 高人出现
4. 秘境发现

## 参考作品
- 《我能看到奇遇提示》
- 《我有一座藏宝阁》
- 《我被诡异封印了》
