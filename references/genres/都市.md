# 都市题材模板

## 题材特征

### 世界观设定
- 现代都市背景
- 商战、职场、豪门
- 都市生活与人情冷暖

### 核心元素
- 主角逆袭
- 家族恩怨
- 商战争夺
- 情感纠葛

### 常见金手指
- 商业头脑
- 家族传承
- 人脉广泛
- 特殊能力

## Strand 比例建议
- Quest线: 50%（事业扩张、商业版图）
- Fire线: 35%（豪门恩怨、情感纠葛）
- Constellation线: 15%（都市阶层、社会百态）

## 爽点类型
1. 商战胜利
2. 打脸对手
3. 抱得美人归
4. 家族崛起
5. 社会地位提升

## 钩子类型
1. 商业对手挑战
2. 家族秘密揭露
3. 情感纠葛
4. 隐藏身份曝光
