# 历史题材模板

## 题材特征

### 世界观设定
- 古代历史背景
- 穿越/重生到古代
- 历史人物/事件

### 核心元素
- 朝堂争斗
- 战争军事
- 商业发展
- 文化传播

### 常见类型
- 穿越历史
- 重生历史
- 架空历史
- 历史同人

## Strand 比例建议
- Quest线: 50%（权力斗争、开疆扩土）
- Fire线: 25%（情感纠葛、君臣关系）
- Constellation线: 25%（历史设定、朝代格局）

## 爽点类型
1. 朝堂翻云覆雨
2. 战争以少胜多
3. 经济发展革新
4. 人才招揽
5. 收复失地

## 钩子类型
1. 朝堂危机
2. 外敌入侵
3. 王朝更迭
4. 皇位争夺
