# 系统流题材模板

## 题材特征

### 世界观设定
- 现代都市或异世界
- 系统作为金手指核心
- 任务驱动剧情

### 核心元素
- 系统发布任务
- 属性面板/技能树
- 任务奖励/兑换商店
- 副本挑战

### 常见系统类型
- 超级商场系统
- 任务系统
- 直播系统
- 养成系统
- 签到系统

## Strand 比例建议
- Quest线: 65%（任务、副本挑战）
- Fire线: 20%（直播互动、粉丝经济）
- Constellation线: 15%（系统设定、世界观）

## 爽点类型
1. 系统升级（新功能解锁）
2. 任务完成奖励
3. 属性暴涨
4. 技能秒杀
5. 粉丝暴涨

## 钩子类型
1. 新任务发布
2. 隐藏成就解锁
3. 限时副本开启
4. 系统升级
