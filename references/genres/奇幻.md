# 奇幻题材模板

## 题材特征

### 世界观设定
- 西方奇幻世界
- 魔法与剑
- 种族林立

### 核心元素
- 魔法修炼
- 种族战争
- 勇者冒险
- 王国纷争

### 常见金手指
- 魔法天赋
- 神器认主
- 召唤系统

## Strand 比例建议
- Quest线: 55%（冒险、战斗）
- Fire线: 20%（种族羁绊）
- Constellation线: 25%（世界观）

## 爽点类型
1. 魔法突破
2. 斩杀魔王
3. 神器获得
4. 种族崛起
5. 王国建立

## 钩子类型
1. 魔王觉醒
2. 遗迹发现
3. 种族大战
4. 王位争夺
